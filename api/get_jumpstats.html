Introduction

/plugin/versions

GET Returns the latest cs2kz-metamod releases.

POST Notifies the API that a new version of cs2kz-metamod has been released.

/plugin/versions/{version}

GET Returns metadata about the release of a specific cs2kz-metamod version.

/users

GET Returns all users with permissions.

/users/{user_id}

GET Returns the user with the specified ID.

/users/{user_id}/email

PUT Updates a user's email address.

DELETE Deletes a user's email address.

/users/{user_id}/permissions

PUT Update a user's permissions.

/auth/cs2

GET Establishes a WebSocket connection with the requesting CS2 server.

/auth/web

GET Returns information about your current session.

/auth/web/login

GET Login with Steam.

/auth/web/logout

GET Expires your current session immediately.

/servers

GET Returns the most recently approved CS2 servers.

POST Approves a new CS2 server.

/servers/{server}

GET Returns the CS2 server with the specified ID / name.

/servers/{server_id}

PATCH Updates a server's metadata.

/servers/{server_id}/access-key

PUT Generates a new access key for a server and invalidates the old one.

DELETE Deletes a server's access key.

/players

GET Returns CS2 players.

/players/{player}

GET Returns the player with the specified ID / name.

/players/{player_id}/profile

GET Returns a player's profile information.

/players/{player_id}/steam-profile

GET Returns a player's Steam profile.

/players/{player_id}/preferences

GET Returns a player's cs2kz-metamod preferences.

PUT Replaces a player's cs2kz-metamod preferences.

/maps

GET Returns the latest KZ maps.

PUT Approves a new map.

/maps/{map}

GET Returns the map with the specified ID / name.

/maps/{map_id}

PATCH Updates a map in-place.

/jumpstats

GET Returns the latest jumpstats.

/jumpstats/{jumpstat_id}

GET Returns the jumpstat with the specified ID.

/jumpstats/{jumpstat_id}/replay

GET Returns the replay file for a specific jumpstat.

/records

GET Returns the latest records.

/records/{record_id}

GET Returns the record with the specified ID.

/records/{record_id}/replay

GET Returns the replay file for a specific record.

/bans

GET Returns the latest player bans.

POST Bans a player.

/bans/{ban_id}

GET Returns the ban with the specified ID.

DELETE Reverts a ban and creates an unban.

PATCH Updates an active ban.

PHP
<?php
$url = 'https://api.cs2kz.org/jumpstats';
$method = 'GET';
$headers = [
    'Content-Type' => 'application/json',
];

$ch = curl_init();
curl_setopt($ch, CURLOPT_URL, $url);
curl_setopt($ch, CURLOPT_RETURNTRANSFER, true);
curl_setopt($ch, CURLOPT_CUSTOMREQUEST, $method);
curl_setopt($ch, CURLOPT_HTTPHEADER, $headers);

$response = curl_exec($ch);
curl_close($ch);

echo $response;
?>
